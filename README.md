# Portfólio de Projetos AZ-900

Bem-vindo ao meu portfólio de projetos relacionados à certificação AZ-900! Este repositório contém uma coleção de projetos que demonstrem as arquiteturas e soluções criadas utilizando o Azure.

## Estrutura do Repositório

- `projetos/`: contém subdiretórios para cada projeto.
- `imagens/`: contém imagens relacionadas ao Azure.

## Como Navegar

Cada projeto possui uma descrição detalhada, diagramas de arquitetura, arquivos de banco de dados e links para templates e documentações.
